CentOS 7环境搭建

镜像版本
CentOS-7-x86_64-Minimal-2003.iso

network

CentOS 7 mini版本，默认没有ifconfig，没有ping，联不通外部网络，不能使用yum install安装软件包。首先要解决的就是网络环境问题。

  • 1.首先进入到网络配置文件所在目录

      cd /etc/sysconfig/network-scripts

    可以看到，有ifcfg-ens33、ifcfg-lo

  • 2.修改ifcfg-ens33配置文件

      vi ifcfg-ens33

    ONBOOT=no 改为 ONBOOT=yes

  • 3.重启网络服务

      systemctl restart network
  或者
  service network restart

  • 4.安装ifconfig、ping等软件包

      # 先搜索，可以搜索到 net-tools.x86_64
  yum search ifconfig
  #net-tools.x86_64 : Basic networking tools
  
  # 安装网络工具包（net-tools应该是包含了ifconfig、ping等命令）
  yum install net-tools.x86_64

    可以查看安装情况，可以看到ifconfig没有，net-tools有

      rpm -qa|grep ifconfig
  rpm -qa|grep net_tools

    试一下，ifconfig、ping了，都可以使用了

      ifconfig
  ping www.baidu.com

base tool

  • 1.自带软件

    • vi（vim minimal）：文本编辑器
    • ssh：有了这个服务，就可以用xshell连接本地的linux系统了，方便操作
    • 压缩软件： tar，bzip,bzip2
    • 防火墙： iptables、firewalld
    • 服务命令： systemctl

  • 2.其他工具软件

    • zip、unzip

      Linux 下面默认会有 tar 程序。如果是解压缩 zip 包，就需要另行安装。

        yum install zip.x86_64 unzip.x86_64
  apt-get install zip unzip

    • vim：文本编辑器

      更加友好的 vim。需要另行安装。字体颜色（注释、key、value各不相同），更易区分

        yum install vim

    • wget：下载工具

        yum install wget

其他软件，根据需要自行下载。
